🎨 直島を彩る関連作品

島内に点在する、世界的なアーティストによる代表的な作品をご紹介します。

草間 彌生

赤かぼちゃ

宮浦港の緑地に設置された、直島の玄関口を象徴する作品。「太陽の赤い光を宇宙の果てまで探して、直島の海の中で赤かぼちゃになった」というコンセプトのもと制作されました。内部に入ることもできます。

ウォルター・デ・マリア

タイム・タイムレス・セルフ

地中美術館にある巨大なインスタレーション作品。直径2.2メートルの球体と、金箔を施した27個の木製彫刻が階段状の空間に配置されています。自然光の変化によって、空間全体の表情が劇的に変わります。

藤本 壮介

直島パヴィリオン

「28番目の島」というコンセプトで作られた浮島のような建築作品。約250枚のステンレス製の三角形のメッシュで構成されており、夜間は内側からのライトアップによって幻想的な姿を見せます。

安藤 忠雄

建築そのものがアート

直島の多くの施設を設計した安藤忠雄氏の建築は、それ自体が重要な作品です。コンクリート打ち放しの壁面と、切り取られた空や光のコントラストは、直島の風景に欠かせない要素となっています。

Visual Arts / Key

Summer Pockets (REFLECTION BLUE)

瀬戸内海の架空の島「鳥白島（とりしろじま）」を舞台にしたアドベンチャーゲーム。直島のフェリーターミナルや周辺の風景が作中の背景モデルとして描かれており、多くのファンが「聖地巡礼」として直島を訪れるきっかけとなっています。

アニメ化プロジェクト

アニメ版：Summer Pockets

原作ゲームの感動的なストーリーをアニメ化。瀬戸内の美しい夏、ノスタルジックな島の日常、そして島に伝わる伝承を巡る物語が描かれます。アニメを通じて、直島を含む瀬戸内の島々の美しさがより広く知られるようになりました。
